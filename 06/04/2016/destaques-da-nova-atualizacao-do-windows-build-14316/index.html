Destaques da nova atualização do Windows (Build 14316)

Fala pessoal.

Hoje (06/04/2016) foi disponibilizado pela Microsoft para membros Windows Insider a nova atualização com os recursos divulgados no evento Build 2016. Para os demais usuários, esta atualização será liberada provavelmente em Julho no Windows 10 Anniversary Update.

Dentro as dezenas de recursos divulgados, eu destaco os que mais me agradaram.

1- Bash nativo do Ubuntu no Windows

Agora é possivel rodar nativamente o bash no windows. Isso é incrivel para nós desenvolvedores que usamos o Windows como sistema operacional. Teremos praticamente o linux inteiro via command line no Windows (não é VM!). Comandos como apt-get, ssh, sudo etc. Segundo a Microsoft, trata-se de um sub-sistema Linux que rodara dentro do Windows. Para ativar, você precisa estar com o Windows configurado para Desenvolvedor, e depois ativar o recurso no gerenciamento de Programas e Recursos do Windows. Mais informações aqui.

 

2- Extensões para o Edge

É de se notar o esforço que a Microsoft vem tendo para entregar um navegador a altura dos grandes Chrome e Firefox. E o que a ocmunidade tanto questionou desde o lançamento do Edge, foi sobre as extensões. Com essa nova atualização, foi liberado a opção para instalar extensões. É claro que ainda não temos disponível muitas opções como nos outros navegadores, mas é uma questão de tempo para serem desenvolvidas.

Gostaria de deixar um comentário pessoal sobre o Edge. Durante alguns dias, eu deixei o Edge como meu navegador principal, e utilizei ele para trabalho. O que pude notar foi um navegador rapido, mais rápido mesmo que o próprio Chrome e Firefox, comparando no Gerenciador de Tarefas o consumo de memória dele foi menor. As ferramentas para desenvolvedor se asemelham aos da ultima versão do Internet Explorer, com um visual mais agradavel. Neste ponto não deixa nada a desejar comparado aos outros navegadores.

3- Recursos multi-dispositivos do Cortana

O assistente pessoal do Windows esta cada vez mais inteligente_, _agora com recursos multi-dispositivos o Cortana manterá você conectado onde você estiver. É importante lembrar que nem todos os recursos do Cortana estão disponíveis em Português. Segue algumas novidades.

  • Notificação de bateria
    O cortana irá informar você sobre a bateria fraca dos seus dispositivos. Seja celular, notebook ou table. Por exemplo, se a bateria do seu celular estiver fraca, Cortana irá te avisar no computador lembrando-o de carrega-lo.
  • Localização do Celular
    Se você perder seu celular, você poderá perguntar ao Cortana sobre a localização do seu dispositivo.
  • Compartilhar mapas e direções entre dispositivos
    Atualmente gastamos alguns esforços para conectar alguns pontos entre nossos dispositivos. Exemplo: mandar um e-mail com um link para abrir no celular, ou pesquisar o endereço de algum lugar. Agora é possivel compartilhar essas informações habilitando os dados de navegação entre dispositivos com o Cortana.
    Como eu havia dito, existem uma dezenas de novidades já anunciada pela Microsoft, transformando a maneira como utilizamos nossos dispositivos. Se você esta ansioso pra testar, ingresse no programa Windows Insider, e você poderá baixar a versão de teste Build 14316.

Eu não recomendo fazer isso no seu computador de trabalho, pois podem ocorrer instabilidades. A atualização oficial provavelmente sairá em Julho de 2016.

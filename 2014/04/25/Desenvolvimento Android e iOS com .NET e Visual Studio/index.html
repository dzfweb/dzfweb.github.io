Desenvolvimento Android e iOS com .NET e Visual Studio

Desenvolvimento mobile sempre foi um grande desafio para muitos desenvolvedores devido a suas limitações em opões de plataformas para trabalhar, seja para Android ou iOS. Deve-se estudar ambientes de produção diferentes, Eclipse, objective C, java, etc.

O Xamarin é uma grande e robusta ferramenta paga (e muito cara por sinal, com a licençaa de uso mais barata de U$299,90 por ano) para desenvolvedores familiarizados com o .NET Framework , sendo possível utilizar o imenso repertório de bibliotecas que este incrível framework da Microsoft disponibiliza com c#, para desenvolver aplicações mobiles. O Xamarin conta com uma documentação gigantesca para estudo, com exemplos de diversos recursos, projetos prontos para download, comunidade para debate com outros desenvolvedores especializados no assunto.

Se você é um desenvolvedor web .NET e deseja se aventurar por uma novo cenário de desenvolvimento, no caso o Desenvolvimento Mobile, uma das áreas mais procuradas para projetos em crescente a cada ano devido ao avanço tecnológico móvel, esta é uma ferramenta profissional muito boa de se estudar.
